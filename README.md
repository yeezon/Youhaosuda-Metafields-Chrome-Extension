## Welcome to Youhaosuda Metafields Chrome Extension

* 预览版，需要的请自行研究和开发；
* 界面用的是 Vue.js，教程可到官方网站查看；
* 修改`assets/main.coffee`里的`_oDefaultMeta`对象与`mainTpl`模板绑定相关数据即可，也可自行开发修改。


使用步骤：

1. 应用插件 - 私有应用 - 创建私有应用 - 查看 - 获取 API Key 和 Shared Secret
2. 设置插件 绑定店铺
